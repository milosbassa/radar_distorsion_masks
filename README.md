# SAR distorsion masks of Republic of Serbia

This repository contains SAR geometric distortion masks and spatial distribution maps derived from Sentinel-1 data processing using ESA SNAP software.

## Dataset content
- 6 raster-based distortion masks
- 12 spatial distribution maps (mean and sum)

## Spatial reference
- CRS: WGS 1984 (EPSG:4326)
- Resolution: 10 m
- Format: GeoTIFF
